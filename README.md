# Campus Plant Classification

## Folders
- `data/` dataset (`train/` and `test/`)
- `scripts/` main programs  
  - `train_V3.py` train the model  
  - `evaluate.py` evaluate the model  
  - `app.py` run the GUI
- `outputs/` saved models, logs, metrics, example images
- `results/` feature search and cross-validation results

## Run
Open and run:
- `scripts/app.py` for the GUI
- `scripts/train_V3.py` for training
- `scripts/evaluate.py` for evaluation




start command

 Windows powershell
python scripts/train_V3.py;
python scripts/evaluate.py;
python scripts/app.py

Mac / Linux
python3 scripts/train_V3.py 
python3 scripts/evaluate.py 
python3 scripts/app.py
